# 🦠 COVID-19 Dashboard (Power BI)
This is a Power BI dashboard project created to analyze and visualize COVID-19 data. It helps in understanding the spread of the virus using interactive charts and filters.

## 📁 Project File
- PBL.pbix – Main Power BI dashboard file

## 📊 Features
- Interactive dashboard for COVID-19 data  
- Shows:
  - Confirmed cases  
  - Deaths  
  - Recoveries  
- Country-wise analysis  
- Trend analysis over time  
- Easy filtering using slicers  

## 📌 Insights
- Helps track COVID-19 spread across different regions  
- Compares cases, recoveries, and deaths  
- Shows trends and patterns over time  

## 🛠️ Tools Used
- Power BI  
- Data Analysis  
- Data Visualization  

## ▶️ How to Use
1. Download the `.pbix` file  
2. Open it in Power BI Desktop  
3. Explore the dashboard  

## 📌 Requirements
- Power BI Desktop
